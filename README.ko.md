# Doguri (도구리)

개발자와 비개발자 모두를 위한 유용한 웹 도구 모음입니다.

> 매번 검색하고 찾아서 사용하는 번거로움을 없애고자 만든 도구 모음으로, 복잡한 기능보다 실용성과 접근성에 중점을 두었습니다.

[English](./README.md)

## 주요 기능 (28개 도구)

### 텍스트
| 도구 | 설명 |
|------|------|
| JSON 정리 | JSON 포맷팅, 검증, 압축 |
| 대소문자 변환 | camelCase, PascalCase, snake_case 등 변환 |
| 공백 제거 | 불필요한 공백 및 줄바꿈 제거 |
| 콤마 추가 | 숫자에 천 단위 콤마 추가 |
| 정규식 테스터 | 정규식 테스트, 검색, 치환 |
| Diff 비교 | 두 텍스트 차이점 비교 및 시각화 |
| 마크다운 <-> HTML | 마크다운과 HTML 상호 변환 |
| 글자수 계산 | 글자/단어/문장/문단 수 실시간 계산 |

### 인코딩
| 도구 | 설명 |
|------|------|
| Base64 | 텍스트/파일 인코딩 및 디코딩 |
| URL 인코딩 | URL 안전 문자열 변환 |
| JWT 디코더 | JWT 토큰 디코딩 및 만료 정보 확인 |

### 보안
| 도구 | 설명 |
|------|------|
| 해시 생성기 | MD5, SHA-1, SHA-256, SHA-512 해시 생성 |
| UUID 생성기 | UUID v4 및 커스텀 길이 ID 생성 |
| 비밀번호 생성기 | 안전하고 강력한 비밀번호 생성 |

### 변환
| 도구 | 설명 |
|------|------|
| 단위 변환기 | 길이, 무게, 온도, 면적, 파일 크기 등 변환 |
| 환율 계산 | 실시간 환율 조회 및 통화 변환 |
| 진법 변환기 | 2진수, 8진수, 10진수, 16진수 간 변환 |
| 이미지 형식 변환 | PNG, JPG, WebP 간 이미지 형식 변환 |

### 시간
| 도구 | 설명 |
|------|------|
| 세계 시간 (한국 기준) | KST 기준 전 세계 주요 도시 시간 확인 |
| 세계 시간 (UTC 기준) | UTC 시간, 타임스탬프, ISO 8601 변환 |

### 계산기
| 도구 | 설명 |
|------|------|
| 날짜 계산기 | D-day 계산, 날짜 차이, 날짜 더하기/빼기 |
| 나이 계산기 | 만 나이, 연 나이, 다음 생일까지 남은 일수 |

### 일상
| 도구 | 설명 |
|------|------|
| 랜덤 추첨기 | 목록/숫자 범위에서 랜덤 선택 |

### 개발자
| 도구 | 설명 |
|------|------|
| QR 코드 생성기 | 텍스트/URL을 QR 코드로 변환 및 다운로드 |
| Cron 생성기 | Cron 스케줄 생성 및 검증 |
| 색상 피커 | HEX, RGB, HSL 형식 간 색상 변환 |
| .env 편집기 | 환경 변수 파일 정리, 중복 검사, 마스킹, 정렬 |
| HTTP 상태 코드 | HTTP 상태 코드 의미 및 사용 사례 검색 |

## 기술 스택

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- React Router 7
- Zustand (상태 관리)
- Vitest (테스트)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 테스트 실행
npm test

# 배포 (GitHub Pages)
npm run deploy
```
