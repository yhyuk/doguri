# Changelog

Doguri 프로젝트의 버전별 주요 변경사항입니다.

---

## [1.1.0] - 2026-04-08

### 새로운 기능
- 5개 신규 도구 추가: JWT 디코더, 진법 변환기, 이미지 형식 변환, .env 편집기, HTTP 상태 코드 사전
- 홈 화면 추가: 카테고리별 카드 그리드로 전체 도구를 한눈에 탐색
- 홈 화면 검색 및 카테고리 필터 기능

### 개선사항
- 단위 변환기에 파일 크기(Data) 카테고리 추가 (B/KB/MB/GB/TB/PB + KiB/MiB/GiB/TiB)
- 카테고리별 아이콘 및 색상 체계 적용
- 사이드바 로고 클릭 시 홈 화면으로 이동

---

## [1.0.2] - 2026-04-03

### 새로운 기능
- 7개 신규 도구 추가: 글자수 계산, 날짜 계산기, 나이 계산기, 랜덤 추첨기, 비밀번호 생성기, QR 코드 생성기, 색상 피커
- 새 카테고리 추가: 계산기, 일상

### 개선사항
- 텍스트 카운터 UI 개선 (트위터 제한 기능 제거)

### 새로운 기능
- Cron 생성기 추가

### 개선사항
- 실시간 시간 업데이트 개선
- 타임라인 버그 수정
